# Rhythm Trainer

A React + Tone.js rhythm-study tool for students practicing clapping or vocal syllables.

## Run

```bash
npm install
npm run dev
```

## Modes

- **Demo**: count-in, metronome, and full generated rhythm.
- **Practice**: count-in and metronome only. The pattern remains visible and highlighted.

## Study modes

- **Clapping**: short percussive attacks, no sustained notes.
- **Voice / Ta**: includes sustained notes and vocal syllables.
